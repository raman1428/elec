Basic Electronics Components & Instruments

Unit I: Electrical Signals and Parameters

  • Electrical Signals: Characteristics of AC/DC signals, waveform properties
  • Parameters:
    • Current (I): Electron flow measurement
    • Voltage (V): Potential difference
    • Wattage (P): Power calculations (P=VI)
  • Components:
    • Resistance: Ohm's Law, color coding
    • Capacitor: Types (ceramic, electrolytic), capacitance values
    • Inductor: Magnetic energy storage, henry measurement
    • Transformer: Step-up/down原理
    • Switches: SPST, SPDT configurations
    • Semiconductor: Diodes, transistors basics

Unit II: Circuit Connections

  • Serial Connections:
    • Resistors: R_total = R1 + R2 + ... + Rn
    • Capacitors: 1/C_total = 1/C1 + 1/C2 + ...
    • Inductors: L_total = L1 + L2 + ...
  • Parallel Connections:
    • Resistors: 1/R_total = 1/R1 + 1/R2 + ...
    • Capacitors: C_total = C1 + C2 + ...
  • Phase Relationships:
    • Lead Connection: Capacitive circuits (current leads voltage)
    • Lag Connection: Inductive circuits (current lags voltage)

Unit III: Measurement Instruments

  • Voltmeter:
    • Range: Selection of appropriate voltage scales
    • Resolution: Minimum detectable voltage change
  • Ammeter:
    • Shunt resistance usage
    • Loading effect prevention
  • Multimeter:
    • Digital vs analog types
    • Measurement modes (Ω, V, A)
    • Continuity testing

Unit IV: Signal Visualization & Applications

  • Signal Visualization:
    • Waveform analysis (sine, square, sawtooth)
    • Frequency and amplitude measurement
  • Oscilloscope:
    • CRT vs digital storage oscilloscopes
    • Timebase and trigger controls
    • Probe calibration
  • Applications:
    • Circuit debugging
    • Signal integrity analysis
    • Component characteristic plotting
